# Installation

**Gradle**
```gradle  
implementation 'io.github.dzmitry-lakisau:month-year-picker-dialog:1.0.0'  
```

**Maven**
```xml  
<dependency>  
	<groupId>io.github.dzmitry-lakisau</groupId>  
	<artifactId>month-year-picker-dialog</artifactId>  
	<version>1.0.0</version>  
</dependency>  
```
